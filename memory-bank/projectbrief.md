# 项目简介

## 项目概述
Virtual Avatar 是一个基于 Web 的 3D 虚拟数字人交互系统，结合了实时语音对话、3D 场景渲染和 AI 对话能力。用户可以通过语音与虚拟角色进行自然对话，系统会实时处理音频、生成响应并播放语音回复。

## 核心目标
1. **沉浸式 3D 体验**：在浏览器中提供高质量的 3D 虚拟环境
2. **实时语音交互**：支持用户通过麦克风与虚拟角色对话
3. **AI 智能对话**：集成大型语言模型提供智能回复
4. **跨平台访问**：通过 Web 技术实现无需安装的访问体验

## 主要功能
- **3D 场景渲染**：使用 Three.js 渲染虚拟房间和角色
- **语音输入处理**：实时接收和处理用户语音
- **AI 对话处理**：通过 SiliconFlow API 进行语音识别和文本生成
- **语音合成输出**：将 AI 回复转换为语音播放
- **交互式 UI**：提供加载状态、对话框和状态显示

## 技术栈
- **前端**：Vue 3 + TypeScript + Vite + Three.js
- **后端**：Node.js + TypeScript + WebSocket
- **AI 服务**：SiliconFlow API（语音识别、LLM、语音合成）
- **部署**：Docker + Docker Compose
- **音频处理**：FFmpeg + Web Audio API

## 项目结构
```
virtual-avatar/
├── web/                    # 前端应用
│   ├── src/
│   │   ├── components/     # Vue 组件
│   │   │   ├── 2D/        # 2D UI 组件
│   │   │   └── 3D/        # 3D 游戏对象和管理器
│   │   ├── views/         # 页面视图

│   └── src/               # 服务器源代码
└── memory-bank/           # 项目记忆库（当前目录）
```

## 关键组件
1. **GameManager**：3D 游戏主控制器
2. **ModelManager**：3D 模型加载管理器
3. **TextureManager**：纹理贴图管理器
4. **SceneManager**：3D 场景管理器
5. **WebSocket 服务器**：处理实时音频通信
6. **AI 处理管道**：语音识别 → LLM → 语音合成
